# NBA Player Intelligence Dashboard

A full-stack web application for searching and viewing NBA player profiles with real-time data integration.

## 🚀 Features
- Search any NBA player by name (case-insensitive)
- View detailed player profiles with team information
- Hybrid search system with intelligent caching
- Responsive, professional UI design

## 🛠 Tech Stack
- **Frontend**: React, CSS3
- **Backend**: Node.js, Express.js
- **API**: Ball Don't Lie NBA API
- **Architecture**: Multi-layer caching system
